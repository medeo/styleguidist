##### font-weight

```jsx
<Span weight="lighter">ceci est un span lighter</Span>
<br/>
<Span>ceci est un span normal (par défaut)</Span>
<br/>
<Span weight="bold">ceci est un span bold</Span>
```

##### color

```jsx
<Span>ceci est un span ebony (par défaut)</Span>
<br/>
<Span color="nevada">ceci est un span nevada</Span>
<br/>
<Span color="aqua">ceci est un span aqua</Span>
<br/>
<span style={{background: "black"}}>
	<Span color="alabaster">ceci est un span alabaster</Span>
</span>
```

##### sizes

```jsx
<Span size="small">ceci est un span small</Span>
<br/>
<Span>ceci est un span medium (par défaut)</Span>
<br/>
<Span size="large">ceci est un span large</Span>
```
